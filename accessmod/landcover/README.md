# Lancover pipeline


Retrieves landcover data from [ESA](https://esa-worldcover.org/en/data-access) WorldCover products.   
The product user manual can be found on the official website [here](https://esa-worldcover.s3.eu-central-1.amazonaws.com/v200/2021/docs/WorldCover_PUM_V2.0.pdf).<br>


Also, data is stored into s3 public bucket, and info can be found [there](https://registry.opendata.aws/esa-worldcover-vito/).<br>
