# Elevation pipeline


Retrieves elevation data from the Copernicus Digital Elevation Model [DEM](https://documentation.dataspace.copernicus.eu/APIs/SentinelHub/Data/DEM.html#filtering-options).   
API Documentation can be found on the official website [here](https://documentation.dataspace.copernicus.eu/APIs/SentinelHub/Process/Examples/DEM.html).<br>


Also, data is stored into s3 public bucket, and info can be found [there](https://registry.opendata.aws/copernicus-dem/) which leads to a readme.md file detailing data processin and structure ([here](https://copernicus-dem-30m.s3.amazonaws.com/readme.html)).<br>
